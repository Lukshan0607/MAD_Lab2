"# MAD_Lab2" 

AIT_ai therapist (Android XML UI Design)
Course Module: IT2010 – Mobile Application Development

Programme: BSc (Hons) in Information Technology (2nd Year)

Institution: Faculty of Computing, SLIIT

Evaluation: Lab Exam 02

Note: This project focuses strictly on Frontend UI design using Android XML to evaluate layout skills, styling, and basic screen navigation as per the specific exam requirements. No backend or actual AI functionalities are implemented.

A modern, empathetic Android application designed to act as a mindful mental health companion. By merging accessible UI design with automated emotional support, the app serves as a private, 24/7 first line of defense for mental well-being.
